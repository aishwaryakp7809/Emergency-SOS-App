# Emergency SOS App

A simple and effective web application that helps users send emergency alerts with live location during critical situations.

## Features
-  Real-time location detection
- Opens location in Google Maps
- One-click SOS alert system
- Clean and responsive UI
- Emergency contact information

## How It Works
1. User clicks the SOS button  
2. System detects current location  
3. Opens Google Maps with live coordinates  
4. Can be extended to send alerts via WhatsApp/SMS  

## Tech Stack
- HTML
- CSS
- JavaScript
